# Motion Surface

A shared layout animation container for list items with smooth background transitions. Features smooth animated background transitions between items, hover support, active state management, and natural physics-based animations powered by Framer Motion.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | List items to wrap |
| `activeIndex` | `number \| null` | `null` | Explicitly active item index |
| `enableHover` | `boolean` | `true` | Enable hover-based activation |
| `activeClassName` | `string` | `"bg-muted"` | Class for active state |
| `hoverClassName` | `string` | `"bg-muted/60"` | Class for hover state |
