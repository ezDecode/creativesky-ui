# Adaptive Tooltip

An intelligent tooltip system that adapts to user intent with confidence-based delays. The tooltip delay decreases as the user hovers over multiple items, showing confidence in their intent to explore tooltips.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `AdaptiveTooltipItem[]` | - | Array of tooltip items |
| `enablePinning` | `boolean` | `false` | Enable keyboard pinning |

### AdaptiveTooltipItem

| Prop | Type | Description |
|------|------|-------------|
| `icon` | `ReactNode` | Icon element to display |
| `label` | `string` | Tooltip text label |
| `hasBadge` | `boolean` | Show notification badge |
