An intelligent tooltip system that adapts to user intent. The tooltip delay decreases as the user hovers over multiple items, showing confidence in their intent to explore.

## Props

**`items`** `AdaptiveTooltipItem[]` — Array of tooltip items to display.

**`enablePinning`** `boolean` — Enable keyboard pinning functionality (default: `false`).

### AdaptiveTooltipItem

**`icon`** `ReactNode` — Icon element to display.

**`label`** `string` — Tooltip text label.

**`hasBadge`** `boolean` — Show a notification badge on the icon.
