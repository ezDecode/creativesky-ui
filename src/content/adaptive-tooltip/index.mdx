An intelligent tooltip system that adapts to user intent. The tooltip delay decreases as the user hovers over multiple items, showing confidence in their intent to explore.

## Installation

<Steps>
  <Step>
    Install the required dependencies:
    ```bash
    npm install framer-motion animejs clsx
    ```
  </Step>
  <Step>
    Copy the component code into your project.
  </Step>
</Steps>
