Animated filter chips with spring-based shared layout transitions. Great for category filters and segmented controls.

## Usage

```tsx
<FilterChips
  filters={["All", "Design", "Development"]}
  activeFilter={active}
  onFilterChange={setActive}
/>
```

## Props

**`filters`** `string[]` — Array of filter labels to display.

**`activeFilter`** `string` — The currently selected filter value.

**`onFilterChange`** `function` — Callback triggered when a new filter is selected.

**`variant`** `"pill" | "outline"` — The visual style of the chips (default: `"pill"`).

**`activeColor`** `string` — Custom background color for the active chip.
