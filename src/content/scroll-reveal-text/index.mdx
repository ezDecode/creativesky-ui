# Scroll Reveal Text

A production-ready, scroll-locked text reveal animation with **dual engine support**. Words are hidden behind animated glassmorphic pill overlays that emerge, glow, then dissolve to reveal the text underneath.

> **Best experienced on desktop.** This component uses scroll-locking animations optimized for larger screens.

**[Open Live Demo →](https://text-box-scroll.vercel.app/)**

---

## Installation

```bash
npm install gsap framer-motion
```

Copy the component files to your project. You'll need `scroll-reveal-text.gsap.tsx` or `scroll-reveal-text.framer.tsx` along with `scroll-reveal-text.module.css`.

---

## Usage

### GSAP Version (Recommended)

```tsx
import { ScrollRevealTextGSAP } from '@/content/scroll-reveal-text/scroll-reveal-text.gsap';

export default function Hero() {
  return (
    <ScrollRevealTextGSAP
      phrase="Your text content here. Each word reveals sequentially."
      title="Welcome"
      highlightWords={["text", "reveals"]}
      primaryColor="#ff6b00"
    />
  );
}
```

### Framer Motion Version

```tsx
import { ScrollRevealTextFramer } from '@/content/scroll-reveal-text/scroll-reveal-text.framer';

export default function Hero() {
  return (
    <ScrollRevealTextFramer
      phrase="Spring-based animations with smooth scroll dampening."
      highlightWords={["Spring", "smooth"]}
      primaryColor="#6366f1"
    />
  );
}
```

---

## Props

**`phrase`** `string` — The text content to be revealed word by word.

**`title`** `string` — Optional heading displayed above the text with gradient styling.

**`highlightWords`** `string[]` — Words to highlight with the accent color. Case-insensitive matching.

**`primaryColor`** `string` — Hex color for highlighted words and glows. Default: `#ff6b00`

**`config`** `object` — Override animation parameters. For GSAP: `leadCount`, `scrollDistance`, `scrubSmooth`. For Framer: `leadCount`, `scrollDistance`, `springStiffness`, `springDamping`.

---

## Config Examples

```tsx
// GSAP - slower, cinematic feel
<ScrollRevealTextGSAP
  phrase="Your text here"
  config={{ leadCount: 10, scrollDistance: 200, scrubSmooth: 3.5 }}
/>

// Framer - bouncier spring
<ScrollRevealTextFramer
  phrase="Your text here"
  config={{ springStiffness: 150, springDamping: 20 }}
/>
```

---

## Animation Phases

**Emergence** — Box fades in at 25% opacity with slight scale.

**Focus** — Box reaches full opacity with glow effect.

**Reveal** — Box dissolves with blur while text fades in.

**Complete** — Text visible, box invisible.

## Accessibility

The component respects `prefers-reduced-motion` by disabling animations. It respects `prefers-contrast: high` with enhanced borders. Decorative boxes use `aria-hidden="true"`.

---

## Troubleshooting

**Scroll not working?** Ensure parent containers don't have `overflow: hidden` — this breaks scroll detection.

**Animation choppy?** Increase `scrubSmooth` for GSAP or `springDamping` for Framer to get smoother motion.

**Text not highlighting?** Ensure words in `highlightWords` match the content. Matching is case-insensitive and uses `includes()`.
